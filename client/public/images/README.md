# Images Directory

Place your images in this directory:

1. **campus-view.jpg** - Main campus building view (St. Joseph's Block)
2. **hostel-building.jpg** - St. Xavier Boy's Hostel building
3. **group-photo.jpg** - Community/alumni group photo

## Image Requirements

- Recommended size: 1920x1080 or higher
- Format: JPG or PNG
- Optimize images for web (use tools like TinyPNG)

## How to Add Images

1. Save your images with the exact filenames:
   - `campus-view.jpg`
   - `hostel-building.jpg`
   - `group-photo.jpg`

2. Place them in: `client/public/images/`

3. The images will be automatically loaded by the Home page

## Fallback

If images are not found, the page will display placeholder icons gracefully.
